一、ryu/ryu目录说明

目录 说明
app 内置的应用
base app_manager.py，其作用是RYU应用的管理中心。用于加载RYU应用程序，接受从APP发送过来的信息，同时也完成消息的路由。
cmd 定义了RYU的命令系统
contrib 主要存放的是开源社区贡献者的代码
controller controller相关的文件，如events.py, ofp_handler.py, controller.py等。其中controller.py中定义了OpenFlowController基类。用于定义OpenFlow的控制器，用于处理交换机和控制器的连接等事件，同时还可以产生事件和路由事件。
lib lib中定义了我们需要使用到的基本的数据结构，如dpid, mac和ip等数据结构
ofproto 两类文件，一类是协议的数据结构定义，另一类是协议解析，也即数据包处理函数文件。
services 完成了BGP和vrrp的实现
tests 单元测试以及整合测试的代码
topology 包含了switches.py等文件，基本定义了一套交换机的数据结构
ryu
mininet基本使用ryu、mininet的基本测试
